#       监督学习(特征值+目标值):(预测)
#           分类: k近邻算法 贝叶斯 决策树与随机森林 逻辑递归 神经网络

#       建立模型:(根据数据类型划分应用种类)  模型:算法+数据
#       1,获取数据
#       2,明确问题
#       3,数据的基本处理:(pd:缺失值,合并表.....)
#       4,特征工程(特征处理),重要
#       5,找到合适的算法去预测
#           不合格 ----->换算法,特征工程再处理
#       6,模型的评估,判定效果
